# Playwright Automation Framework

Welcome to the central repository for the Playwright learning path. This project houses daily lectures, demonstrations, and structured lab practicals.

## 📁 Repository Structure

The project is organized by daily modules to keep learning sequential and structured:

*   **`instructor/`**: Code demos, architectural setups, and reference solutions built during lectures.
*   **`labs/`**: Practical assignments, exercises, and custom test scenarios designed for hands-on practice.

## 🛠️ Tech Stack & Dependencies

*   **Language**: TypeScript / JavaScript
*   **Automation Framework**: Playwright
*   **Execution Engine**: `ts-node` (On-the-fly TypeScript execution)
*   **Package Manager**: `npm`
